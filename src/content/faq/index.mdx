# 常见问题解答

## 一般问题

### 什么是AI IDE?

AI IDE是一个集成了先进人工智能技术的开发环境，旨在提高开发者的生产力。它能够理解代码意图，提供智能建议，自动完成代码片段，甚至生成完整功能。

### AI IDE适合哪些开发者?

AI IDE适合各种水平的开发者：
- 初学者可以通过AI辅助更快地学习编程
- 中级开发者可以借助IDE提高编码效率
- 高级开发者可以专注于解决复杂问题，减少编写重复代码的时间

### AI IDE支持哪些编程语言?

AI IDE目前支持以下语言:
- JavaScript/TypeScript
- Python
- Java
- C/C++
- Go
- Ruby
- PHP
- Rust
- Swift
- SQL
- 更多语言正在持续添加中

## 技术问题

### AI IDE是如何工作的？

AI IDE结合了先进的机器学习模型和静态代码分析技术：
1. 当您编写代码时，AI会分析上下文和编码模式
2. 基于分析结果，AI提供智能建议和补全
3. IDE还会识别潜在的代码问题并提出改进建议

### AI IDE需要联网才能工作吗？

基本功能可以离线使用，但高级AI功能需要网络连接以访问云端AI模型。我们也提供本地AI模型选项，但功能会相对受限。

### 我的代码会被上传到云端吗？

默认情况下，为了提供最准确的建议，代码片段会被发送到我们的安全服务器进行分析。但我们提供严格的隐私设置，您可以选择仅使用本地AI功能，或者指定哪些项目或文件可以用于云分析。
