# 快速入门指南

欢迎使用AI IDE！本指南将帮助您快速上手，充分利用我们强大的AI辅助编程功能。

## 安装步骤

### 系统要求

- 操作系统: Windows 10/11、macOS 10.15+、Linux (Ubuntu 20.04+)
- 内存: 至少8GB RAM (推荐16GB+)
- 硬盘空间: 至少2GB可用空间
- 处理器: 多核处理器，2GHz以上

### 下载与安装

1. 访问[官方下载页面](https://ai-ide.example.com/download)
2. 选择适合您操作系统的版本
3. 运行安装程序并按照向导完成安装

或者通过命令行安装:

```bash
# 使用npm安装
npm install -g ai-ide-cli

# 安装完成后启动
ai-ide
```
