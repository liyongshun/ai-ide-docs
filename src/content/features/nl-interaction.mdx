# 自然语言交互

AI IDE的自然语言交互功能让您可以用日常语言与IDE交流，描述您的需求，并让AI帮助您实现功能，极大简化了开发过程。

## 用语言描述需求

无需记住复杂的语法或API，只需用自然语言描述您想要实现的功能：

```
// 输入自然语言指令
// "创建一个函数，接收一个日期字符串，返回格式化的日期，格式为YYYY-MM-DD"

// AI理解并生成代码:
/**
 * 将日期字符串格式化为YYYY-MM-DD格式
 * @param {string} dateString - 输入的日期字符串
 * @return {string} - 格式化后的日期字符串
 */
function formatDate(dateString) {
  const date = new Date(dateString);
  
  if (isNaN(date.getTime())) {
    throw new Error('Invalid date string');
  }
  
  const year = date.getFullYear();
  const month = String(date.getMonth() + 1).padStart(2, '0');
  const day = String(date.getDate()).padStart(2, '0');
  
  return `${year}-${month}-${day}`;
}
```

## 提出问题与获取解答

您可以直接问AI关于代码的问题，获取即时解答：

```javascript
// 选中以下代码并询问"这段代码有什么潜在问题？"
const userIds = users.map(u => u.id);
for (let i = 0; i < userIds.length; i++) {
  await processUser(userIds[i]);
}

// AI会回答：
// 1. 代码在循环中使用await，会导致串行执行而非并行执行，可能影响性能
// 2. 没有错误处理机制
// 3. 并提供改进建议：
//    - 使用Promise.all进行并行处理
//    - 添加try/catch进行错误处理
```

## 代码重构和优化

描述您希望如何改进现有代码：

```javascript
// 选中代码并请求
// "将这段代码重构为更简洁的形式，使用现代JavaScript特性"

// 原始代码
function getFullName(user) {
  var firstName = user.firstName;
  var lastName = user.lastName;
  var fullName = firstName + ' ' + lastName;
  return fullName;
}

// AI重构后
const getFullName = ({ firstName, lastName }) => `${firstName} ${lastName}`;
```

      width: window.innerWidth,
      height: window.innerHeight,
    });

  window.addEventListener('resize', handler);
